# Practical 05 - Exception Handling

## Description

Create an array of integers with 5 values, iterate through the loop till 5, and print each index of the array. Wrap the `Console.WriteLine` call in a `try` block and catch `IndexOutOfRangeException`. Under `catch`, print the exception message. In `finally`, print "This is finally block".
